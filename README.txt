## 網站使用說明

本網站是一個結合了番茄鐘和任務計畫表的學習管理工具，旨在提高您的學習或工作效率。所有資料都會自動儲存在您的瀏覽器中。

### 一、番茄計時器 (Pomodoro Timer)

此功能幫助您實踐番茄工作法，保持專注。

- **▶ 開始 / ⏸ 暫停**: 點擊此按鈕來啟動或暫停計時器。
- **↺ 重置**: 將當前的計時器重設回初始狀態（預設為 25 分鐘的專注時段）。這不會重置您已完成的番茄總數。
- **⏭ 跳過階段**: 立即結束目前階段，並進入下一個階段（例如，從「專注時段」跳至「休息時段」）。
- **計時器顯示**:
    - **時間**: 以 `分鐘:秒鐘` 的格式顯示剩餘時間。
    - **階段標籤**: 顯示目前是「🍅 專注時段」還是「☕ 休息時段」。
    - **輪數**: 顯示您正在進行的專注輪次。
- **累積統計**: 頁面底部會顯示您已完成的番茄總數和累積的總專注分鐘數。

### 二、任務計畫表 (Task Planner)

此功能幫助您追蹤需要完成的任務和學習進度。

- **新增任務**:
    - 在「寫一個新任務...」的輸入框中輸入您的任務內容。
    - 點擊 **➕ 新增** 按鈕或直接按下 **Enter** 鍵即可將任務添加到列表中。
- **完成任務**:
    - 點擊任務左側的 **方框 (checkbox)**，即可將該任務標記為「已完成」。
    - 任務文字會被劃上橫線，且統計數據會自動更新。
- **刪除任務**:
    - 點擊任務右側紅色的 **✕** 按鈕，即可永久刪除該任務。
- **進度追蹤**:
    - **統計概數**: 頂部會顯示「總任務」、「已完成」和「達成率 (%)」。
    - **進度條**: 底部的長條圖會視覺化地展示您已完成任務的百分比。
